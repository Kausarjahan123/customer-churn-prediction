# Customer Churn Prediction System

## Project Overview
This project predicts whether a customer will leave (churn) or stay based on their usage and account details using Machine Learning.

---

## ML Concept Used
- Supervised Learning
- Classification (Random Forest)

---

## Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit

---

## Features Used
- Age
- Tenure
- Monthly Charges
- Total Charges
- Contract Type
- Internet Service
